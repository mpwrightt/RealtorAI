### 4. **HomeHarvest** (Python Library)
**Website:** https://github.com/Bunsly/HomeHarvest

**What You Get:**
- Property listings from Zillow, Realtor.com, Redfin
- Property details
- Price history

- Rental listings

**Example Usage:**
```python

    location="San Francisco, CA",
    listing_type="for_sale",
    site_name="zillow"
)
```

**Pros:**
- Completely free
- Works for multiple sites
- Open source

**Cons:**
- Scraping (can break if sites change)
- Rate limiting issues
- May violate ToS

**Status:** ‚ö†Ô∏è **USE CAREFULLY** - Free but may have legal risks

### For Walk Score / Places (Walk Score Alternative):

#### **Google Places API**
**Website:** https://console.cloud.google.com/  
**Free Tier:** ‚úÖ $300 free credits for 90 days  
**Then:** $0.017 per request (so $300 = ~17,600 requests)

**What You Get:**

- Distance calculations
- Walking directions

**Setup:**
```bash

# 4. Create API key

```

**Example:**
```javascript
const response = await fetch(
```

```typescript
export async function searchComparableProperties(params: any) {
```
